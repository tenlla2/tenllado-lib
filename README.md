# ATENLLADO-LIB VITE + REACT LIBRARY

## Installation and Setup

### Project Installation
Node version: v21.6.2
Yarn version: 1.22.21

```shell
# Install
yarn install
```

## Usage

### Run development environment
```
yarn dev
```
### Run unit tests
```
yarn test
```
### Run Storybook
```
yarn storybook
```
